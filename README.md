# 🏦 Bank Management System

A console-based Bank Management System built using **Java, JDBC, MySQL, and OOP**.

## Features

### Customer
- Register
- Login
- Deposit
- Withdraw
- Transfer Money
- Check Balance
- Mini Statement
- Transaction History
- Update Profile
- Change PIN

### Admin
- Login
- View Customers
- View Accounts
- Unlock Accounts
- View Login History
- View Audit Logs

## Tech Stack

- Java
- JDBC
- MySQL
- OOP
- Git
- VS Code

## Database

- customers
- accounts
- transactions
- branches
- beneficiaries
- admins
- login_history
- otp_history
- audit_logs

## Run

1. Execute `sql/setup.sql`
2. Update `DBConnection.java`
3. Run `run.bat`
